network map of climate adaptation projects in Vanuatu

This is a network map of climate adaptation projects in Vanuatu in 2016. Click or hover over an organization or activity to see connections.

This page requires HTML5 canvas and JavaScript.

The code for this infographic is adapted from Zack Grossbart's HBORecycling project on GitHub, which was based on a visualization by Craig Robinson. It's made with PaperJS, underscore.js, and a little JQuery.
